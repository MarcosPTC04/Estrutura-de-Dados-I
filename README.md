## 📌 Descrição

Este repositório apresenta a resolução de cinco exercícios de lógica de programação desenvolvidos nas linguagens Java e Python. O objetivo é praticar conceitos fundamentais como vetores/listas, estruturas de repetição, estruturas condicionais, operadores matemáticos e criação de funções (métodos).

Além disso, o projeto permite comparar como cada linguagem resolve os mesmos problemas utilizando sintaxes diferentes, mas mantendo a mesma lógica.

## 📋 Estrutura do Código

O projeto está organizado em cinco exercícios, cada um implementado tanto em Java quanto em Python:

Ranking de notas – calcula a média das notas, conta quantos alunos ficaram acima da média e identifica a maior nota.

Simulador de caixa eletrônico – determina a menor quantidade possível de cédulas de R$ 50, R$ 20 e R$ 10 para um valor múltiplo de 10.

Validador de senha – verifica se a senha possui pelo menos oito caracteres e ao menos um caractere especial.

Conversor de tempo – transforma segundos em horas, minutos e segundos.

Verificador de elementos distintos – verifica se todos os elementos de um vetor/lista são únicos.

Observações sobre a implementação:

Em Java, há declaração explícita de tipos e uso de métodos como length() e charAt().

Em Python, utiliza-se tipagem dinâmica e funções como len(), com sintaxe mais enxuta.

A lógica de cada problema permanece idêntica nas duas linguagens, reforçando os princípios universais da programação.

Os algoritmos utilizam estruturas de repetição, condicionais, operadores matemáticos e funções/métodos para modularizar o código.
